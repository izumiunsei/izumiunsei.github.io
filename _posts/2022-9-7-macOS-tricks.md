## 截图
截图这一点做得比Windows要好，不需要print screen粘贴大法。  
快捷键：
1. 全屏截图：`shift`+`command`+`3`，**不是F3!!**
2. 自由截图：shift + command + `4`
3. 捕捉窗口、菜单： shift + command + `4` + 空格
4. 一键解决，自由选择：shift + command + `5`
